# Kitsu/VM // Learn Vulkan

This is my personal project to learn Vulkan, but it also contains some crates containing abstractions that I find useful.

## Crates

- `kitty-vulkan`: Contains idiomatic Rust wrappers over Vulkan APIs.
- `kitty-window`: Contains idiomatic Rust wrappers over the native windowing APIs of the supported platforms.
- `kitty-vulkan-sys`: Contains raw FFI bindings to Vulkan APIs.
- `learn-vulkan`: Contains my implementation of the Learn Vulkan tutorial.
